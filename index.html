Privacy and Terms of Service

Personal identification information

We may collect personal identification information about the Users which are available from the Telegram Bot API and the Google Drive API. These include Telegram user id from Telegram Bot API and Email address from Google Drive API.


How we use collected information

  • We collect, save, and use personal identification information like Telegram user id and Email address to run and operate the bot. We securely save your identifiers which will be used later for serving you well. We save and use your Email id to differentiate between your linked Google Drive accounts.
  • The service comprise of creating and modifying of Google Drive files. We only access your data on behalf of you when you instruct us to do so.
  • We follows a standard procedure of using log files. These files log users when they use the bot. All similar services do this. The information collected by log files include telegram user id, telegram username, date and time, actions performed, and URLs / telegram file identifiers only when something goes wrong. The purpose of the information is for analyzing trends, administering the bot, and tracking errors.
